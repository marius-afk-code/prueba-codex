# Registro de fútbol

Aplicación sencilla en Python para registrar estadísticas de partidos de fútbol.

## Qué guarda
- Fecha del partido
- Rival
- Goles
- Asistencias
- Tarjetas

La información se guarda en `partidos.json` en la raíz del proyecto.

## Cómo ejecutar
```bash
python3 app.py
```

## Opciones del menú
1. Registrar partido
2. Ver partidos registrados
3. Salir
